<center><h1 style="font-size: 36px;">安装指南 (3DGoto)</h1></center>




 1. 我这里只用一个版本精简版，要开发版自作mods的去原作那里看

 2. 根据你的Genshin Impact.exe文件的位置，你可能需要修改d3dx.ini文件中的这一行，以指示你自己的安装位置。（游戏.exe，而不是启动器.exe）
